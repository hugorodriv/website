About this website

Welcome to my personal website! I will use it to host and showcase various projects, as well as a sort of digital CV.

The website is programmed directly in HTML and uses TailwindCSS for managing the styling.
I manage the hosting myself in a VPS running Debian that uses NGINX for serving the content.

Each project listed will have a description on how it works more technically, as well as a direct link to the GitHub repository.

Some of the projects are built using tools not necessary for the example in question, the point being for me to learn such tools for when they are actually needed, and to showcase my learning process aswell as the final product.
